Configuration Réseau & Cloud

Configurez l'outil pour travailler en équipe de manière synchronisée.

1. Notifications instantanées (Webhook)

Collez l'URL de votre Webhook (Discord, Slack, Make, Zapier). À chaque modification effectuée par un collègue, vous recevrez une alerte en direct.

2. Identité d'édition

Votre nom, classe ou fonction sera inclus dans les notifications envoyées à vos collègues.

3. Votre propre base Firebase (Recommandé)

Saisissez les paramètres de votre base de données Cloud Firebase pour synchroniser toutes les fiches de manière sécurisée et privée sans passer par les configurations individuelles.

Astuce : Vous pouvez aussi configurer cela directement dans le code source de votre serveur GitHub pour qu'il s'initialise automatiquement pour tout le monde !
